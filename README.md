# Number Kanji

[A kanji drawing game to connect numbers](https://marmooo.github.io/number-icon/).

## Installation

1. `bash install-fonts.sh`
2. `deno --allow-read --allow-write build-svg.js`
3. `deno run --allow-read --allow-write listup.js`

## Build

```
bash build.sh
```
